# 私有化与离线部署

私有化版面向数据不能离开客户网络的场景。部署包含 API 网关、任务队列、CPU 预处理节点、GPU 推理节点和结果存储。

## 最小生产拓扑

- 3 个 API/调度副本，跨故障域分布；
- 3 个队列节点或托管等价能力；
- CPU worker 按文件预处理吞吐扩缩；
- GPU worker 按待处理页数和模型类型扩缩；
- 对象存储需要版本化和服务端加密。

## 离线更新

模型、解析器和 schema 使用签名离线包发布。客户环境先在影子流量上运行回归测试，通过后才切换默认版本。紧急回滚不依赖外网连接。

私有化版不包含自动访问公网 URL 的能力；外部内容需由客户网关先行下载和扫描。
