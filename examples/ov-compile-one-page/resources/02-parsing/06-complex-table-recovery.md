# 复杂表格结构恢复

复杂表格包括合并单元格、多层表头、跨页续表、无边框表格和单元格内列表。本方案先识别行列拓扑，再填充文本，避免仅依赖 OCR 坐标拼接。

## 结构化输出

每个单元格包含行列起点、`row_span`、`col_span` 和原始文本区域。同时输出 HTML 表格与结构化 JSON，Markdown 仅用于不含合并单元格的简单表格。

## 跨页续表

只有同时满足表头相似、列对齐相似和前页未出现明确表尾时，才合并两页表格。对“续表”文字只作加分证据，不作唯一条件。

## 已知风险

低分辨率无边框表格容易被识别为多栏正文。当表格置信度低于 0.65 时，应保留页面截图供下游展示和复核。
