# API 与 SDK 对接指南

Atlas Parse 提供异步任务 API，适合接入上传、云盘同步和批量迁移等链路。

## 标准流程

1. 创建解析任务并传入文件或限时下载地址；
2. 保存返回的 `task_id`；
3. 通过回调或轮询等待任务进入终态；
4. 下载结构化 JSON、Markdown 或页级素材；
5. 使用内容哈希避免重复入库。

## 幂等与重试

创建任务时可传入 `idempotency_key`。网络超时后应使用原 key 重试，不要换新 key 创建重复任务。查询接口建议使用指数退避，最长间隔不超过 30 秒。

## SDK

Python 和 Java SDK 封装了分片上传、回调签名校验和结果分页。其他语言可直接使用 HTTP API；协议不要求调用方引入特定运行时。
