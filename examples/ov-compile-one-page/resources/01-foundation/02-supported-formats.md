# 文件格式与能力矩阵

本文档说明 Atlas Parse 对常见企业文件的处理级别。“完整支持”表示可恢复段落、标题、表格和页面定位；“基础支持”表示仅保证主文本提取。

| 格式 | 支持级别 | 主要能力 | 已知限制 |
| --- | --- | --- | --- |
| PDF | 完整支持 | OCR、版面、表格、页码引用 | 动态 XFA 表单不执行 |
| DOCX | 完整支持 | 标题层级、表格、批注位置 | 宏和外部对象不执行 |
| PPTX | 完整支持 | 页面、文本框、演讲者备注 | 动画时序不保留 |
| XLSX | 基础支持 | 工作表、单元格值、常用公式 | 不重算外部数据源 |
| HTML | 完整支持 | 标题、列表、表格、链接 | 不运行客户端 JavaScript |
| 图片 | 基础支持 | OCR、方向校正、区域定位 | 手写体仅做尝试性识别 |

压缩包作为容器处理，单个内部文件仍遵循上述矩阵。加密文件需由调用方提供密码，否则返回可识别的“需要密码”错误。
