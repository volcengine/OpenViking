# 轨迹：发布前进行事实、结构和措辞三层质检

## 情境

一份结构完整的周报仍在发布前被退回：摘要中的百分比与正文基线不一致，
“彻底解决”也超出了观测周期能够支持的结论。

## 质检步骤

1. 事实层：逐项核对数字、时间、状态和负责人是否在输入材料中有依据。
2. 一致性层：检查摘要、表格和正文对同一事项的状态是否一致。
3. 因果层：区分“同时发生”和“由此导致”，删除证据不足的因果结论。
4. 措辞层：把“彻底解决”“完全稳定”等绝对表述改为观测范围内的结论。
5. 阅读层：确保一句话结论先给结果和主要风险，细节随后展开。

## 结果

修订后所有数字可以回溯，摘要与正文一致；评审者只提出排版建议，没有再
发现事实问题。

## 可复用契约

- 发布前必须同时检查事实准确性和跨章节一致性。
- 结论强度不能超过证据的时间范围、样本范围或覆盖率。
- 最终文本优先准确、清晰、可执行，不追求夸张的“成果感”。
