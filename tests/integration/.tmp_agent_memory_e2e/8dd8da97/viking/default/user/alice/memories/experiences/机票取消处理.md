## Situation
- 用户提出取消已预订的机票的需求

## Approach
- 获取用户指定要取消的机票的预订标识，调用cancel_booking工具提交取消请求
- 若取消成功，告知用户取消结果
- 若取消失败，向用户说明失败原因及对应的可选处理方案，等待用户进一步指令

## Reflect
- 不得在未明确用户要取消的具体机票的情况下调用取消工具
- 不得隐瞒取消失败的真实原因和可选项
