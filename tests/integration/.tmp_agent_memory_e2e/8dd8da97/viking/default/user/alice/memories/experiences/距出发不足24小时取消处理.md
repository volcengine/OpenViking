## Situation
- 用户提交机票取消请求，调用取消工具返回距出发时间不足24小时无法免费取消的失败提示

## Approach
- 明确告知用户取消失败原因是距出发时间不足24小时，不符合航空公司免费取消政策
- 向用户说明两种可选处理方案：支付改签费改期、放弃机票仅退税费
- 等待用户选择，按用户后续指示执行对应操作
- 若用户选择不处理，告知用户该机票将保持原有状态

## Reflect
- 不得未告知用户失败原因直接拒绝取消请求
- 不得私自替用户选择改期或放弃的处理方案
- 不得遗漏可选方案的说明
- 不得在用户明确知晓退改政策的场景下重复说明政策内容
